# M.A.P.A. - Programação Front-End 🚀

Este repositório contém o código-fonte do meu Portfólio Profissional, desenvolvido como atividade prática (M.A.P.A.) para a disciplina de **Programação Front-End**. O objetivo do projeto foi integrar as três tecnologias fundamentais da web para criar uma interface moderna, acessível e totalmente responsiva.

## 🛠️ Tecnologias Utilizadas & Aprendizados

* **HTML5 Semântico:** Estruturação de 4 páginas interligadas utilizando tags semânticas (`<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`) para garantir melhor acessibilidade e SEO básico.
* **CSS3 Moderno:** Aplicação de conceitos de *Flat Design*, Flexbox para o alinhamento do menu e **CSS Grid** para a organização responsiva dos cards de projetos.
* **Design Responsivo:** Uso de *Media Queries* para adaptar o layout perfeitamente a dispositivos móveis e desktops.
* **JavaScript Puro (Vanilla JS):** Desenvolvimento de recursos interativos (saudação baseada no horário do sistema e botão para alternar o tema/Dark Mode) e lógica para a **validação completa do formulário de contato** no lado do cliente.

## 📂 Estrutura do Projeto

* `index.html` - Página Inicial (Apresentação e Boas-Vindas)
* `sobre.html` - História Profissional e Competências (com foto de perfil)
* `projetos.html` - Grade com estudos de caso e projetos
* `contato.html` - Formulário de contato validado
* `style.css` - Identidade visual, variáveis e responsividade
* `script.js` - Inteligência e comportamento interativo do site
